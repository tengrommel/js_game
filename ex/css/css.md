# 常见的选择器和伪元素

## 属性选择器：
    
    [属性名称]： 单独选择某个属性
    [属性名称=属性值]： 选择这个属性具体的某个值的元素

## 子元素选择器

- 第几个元素语法
    
    nth-child(1)
    
- 最后一个子元素语法

    last-child
